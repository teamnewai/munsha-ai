## الوكلاء والمخطّط
1. **نور AI — السكرتيرة الذكية** — تقرير يومي/أسبوعي · توجيه بالكلمات · صوت عربي · تفعيل `aiNarrative()` (ثغرة P0).
2. **CEO AI** — ملخّص تنفيذي · مقارنة أداء الأقسام · كشف الشذوذ (owner فقط).
3. **CFO AI** — أرباح وخسائر تلقائية · تنبؤ تدفّق نقدي · توافق ZATCA/VAT.
4. **HR AI** — إجازات · تقييمات · امتثال نظام العمل السعودي.
5. **Legal AI** — مراجعة عقود · مواعيد تنظيمية · مصفوفة RACI.
6. **Marketplace AI** — مطابقة المزوّدين · تحليل العروض · كشف الاحتيال.
7. **AI Board Room (غرفة القرار)** — تعاون الوكلاء عبر ناقل أحداث: Sales→Property→CFO→Legal.
8. **AI Memory Layer** — ذاكرة قصيرة/طويلة/تنظيمية/مشتركة.
9. **حوكمة الوكلاء** — هرم تصعيد: تنفيذ تلقائي / موافقة حسب الحدّ / إيقاف عند الغموض.
