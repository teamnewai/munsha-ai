## البوابات
1. **المستأجر** — عقده · فواتيره ودفعها · طلبات صيانة · مستندات · تجديد/إخلاء.
2. **المالك** — محفظة عقاراته · صافي الدخل/عقار · موافقة صيانة >2000 · تقارير.
3. **عضو HOA** — رسوم · إعلانات · تصويت · حجز مرافق · محاضر.
4. **مزوّد الخدمة** — طلبات واردة · عروض · أعمال معتمدة · إثباتات · تقييم.
5. **مشتري السوق** — تصفّح · طلب · مراجعة عروض · طلبات · تقييم.
6. **المستثمر** — لوحة محفظة · ROI · تحليل سوق · توزيعات.
7. **الجهات الحكومية** — قراءة فقط لبيانات الامتثال · ZATCA · تراخيص.
8. **White-label** — شعار/ألوان/نطاق العميل · إخفاء هوية مُلكي.

## مبدأ الأمان
كل بوابة تصل بياناتها المُنطاقة فقط عبر RLS — لا تتجاوز حدود org/party.
