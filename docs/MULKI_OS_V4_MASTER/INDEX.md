## اصطلاحات

- اللغة الأساسية: العربية (RTL)؛ المصطلحات التقنية وأكواد SQL بالإنجليزية.
- العملة الأساسية: SAR؛ التخزين الزمني: UTC.
- كل اسم جدول/عمود snake_case إنجليزي؛ العزل بـ `org_id` + RLS.
- **REOS:** المنصّة تُسجّل المعاملات ولا تحوز الأموال أبداً.
